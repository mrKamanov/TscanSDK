Сканер тестов - Инструкция по установке
==============================

1. Убедитесь, что у вас 64-битная версия Windows
2. Запустите файл "Сканер тестов.exe"
3. При первом запуске Windows может показать предупреждение безопасности - нажмите "Подробнее" и "Выполнить в любом случае"
4. Дождитесь запуска приложения (это может занять несколько секунд)
5. В браузере откроется интерфейс приложения

Системные требования:
- Windows 7/8/10/11 (64-bit)
- 4 GB RAM (рекомендуется)
- 500 MB свободного места на диске
- Подключенная веб-камера (для функций сканирования)

При возникновении проблем:
- Убедитесь, что порт 5000 не занят другими приложениями
- Проверьте, что антивирус не блокирует приложение
- Для сканирования через телефон, убедитесь что телефон подключен к той же Wi-Fi сети
