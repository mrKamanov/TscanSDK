## 🔬 Принцип работы

### Алгоритм распознавания

Test Vision AI использует многоступенчатый алгоритм обработки изображений для точного распознавания ответов:

1. **Предварительная обработка**
   - Адаптивная коррекция яркости и контраста
   - Устранение шумов и артефактов
   - Нормализация размера изображения

2. **Определение области ответов**
   - Детекция границ бланка
   - Коррекция перспективных искажений
   - Выделение области с ответами

3. **Сегментация ответов**
   - Разбиение на индивидуальные ячейки
   - Анализ заполненности каждой ячейки
   - Определение отмеченных вариантов

4. **Анализ результатов**
   - Сопоставление с правильными ответами
   - Подсчет баллов и статистики
   - Формирование отчета

### Технологии и библиотеки

🔧 **Backend**
- **Python** - основной язык разработки
- **OpenCV** - компьютерное зрение и обработка изображений
- **NumPy** - математические вычисления и работа с массивами
- **Flask** - веб-сервер и API

🎨 **Frontend**
- **HTML5/CSS3** - современная верстка
- **JavaScript** - интерактивность интерфейса
- **Socket.IO** - обмен данными в реальном времени
- **Chart.js** - визуализация данных

💾 **Хранение данных**
- **JSON** - формат хранения результатов
- **Excel** - экспорт отчетов
- **Base64** - кодирование изображений

## 🚀 Быстрый старт

### Предварительные требования

- Python 3.8+
- pip
- Веб-камера (для сканирования в реальном времени)

### Установка

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/mrKamanov/TscanSDK.git
cd TscanSDK
```

2. **Создайте виртуальное окружение**
```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
```

3. **Установите зависимости**
```bash
pip install -r requirements.txt
```

4. **Запустите приложение**
```bash
python app.py
```

## 📖 Документация

### 🎥 Сканирование в реальном времени

Мощный инструмент для мгновенной проверки тестов через камеру устройства.

#### Управление камерой
- 🟢 **Включить камеру** - Запуск видеопотока
- 🔴 **Выключить камеру** - Остановка записи
- ⏸️ **Стоп-кадр** - Фиксация изображения
- ➕ **Добавить в отчет** - Сохранение результатов

#### Настройка изображения
- 🔆 **Яркость** - Регулировка освещенности
- 🎨 **Контраст** - Настройка четкости
- 🎯 **Резкость** - Улучшение детализации
- 🌈 **Насыщенность** - Корректировка цветов

### 📦 Пакетная обработка

Эффективное решение для проверки большого количества тестов.

#### Процесс работы
1. 📤 Загрузка изображений (drag & drop или выбор файлов)
2. ⚙️ Настройка параметров теста
3. ▶️ Запуск обработки
4. 📋 Просмотр результатов
5. 💾 Сохранение в отчет

### 📊 Аналитика и отчеты

Продвинутые инструменты анализа результатов.

#### Возможности
- 📈 Графики успеваемости

- 📊 Распределение оценок
- 📑 Экспорт в Excel

### 📝 Требования к бланкам

#### Оптимальные условия
