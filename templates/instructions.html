Общая информация

Приложение предназначено для автоматической проверки тестовых работ. Система распознает отмеченные ответы и сравнивает их с правильными, формируя подробный отчет о результатах.

Сканирование в реальном времени

Порядок работы:

  1. Включите камеру, нажав кнопку "Включить камеру"
  2. Расположите тестовый бланк в поле зрения камеры
  3. При необходимости настройте параметры изображения (яркость, контраст и т.д.)
  4. Создайте стоп-кадр для фиксации результата
  5. Проверьте правильность распознавания
  6. Отправьте результат в отчёт

Режимы отображения камеры

В приложении доступна функция переключения режима отображения камеры, которая особенно полезна при работе с тестами, содержащими большое количество вопросов:

Обычный режим

Показывает полное изображение с камеры. Удобен для общего контроля и позиционирования бланка.

Режим наложения: ВЫКЛ

Режим наложения

Накладывает результаты распознавания поверх изображения с камеры, что упрощает проверку правильности распознавания ответов в тестах с большим количеством вопросов.

Режим наложения: ВКЛ
Совет

Используйте переключение режимов для удобной проверки тестов разного формата. Режим наложения особенно эффективен при работе с длинными тестами, так как позволяет лучше видеть результаты распознавания.

Важно

Для наилучшего результата распознавания:

  • Убедитесь, что бланк хорошо освещен
  • Избегайте бликов и теней на бланке
  • Располагайте бланк параллельно камере
  • Используйте настройки камеры для улучшения качества изображения

Работа с бланком ответов

Важно: Для корректной работы сканера необходимо использовать правильный формат бланка ответов.

Пример бланка ответов:
Скачать шрифт Скачать бланк

Требования к бланку:

  • Оптимальное количество вопросов: от 5 до 20
  • Поддерживаются только вопросы с единичным ответом
  • При разном количестве вариантов ответов необходимо стереть неиспользуемые ячейки
  • При изменении количества вопросов нужно соответственно изменить количество строк
  • Важно обозначить границу таблицы для корректного распознавания
