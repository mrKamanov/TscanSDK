Приложение предназначено для автоматической проверки тестовых работ. Система распознает отмеченные ответы и сравнивает их с правильными, формируя подробный отчет о результатах.

Сканирование в реальном времени

Порядок работы:

  1. Включите камеру, нажав кнопку "Включить камеру"
  2. Расположите тестовый бланк в поле зрения камеры
  3. При необходимости настройте параметры изображения (яркость, контраст и т.д.)
  4. Создайте стоп-кадр для фиксации результата
  5. Проверьте правильность распознавания
  6. Отправьте результат в отчёт

Режим наложения: ВКЛ
Совет

Используйте переключение режимов для удобной проверки тестов разного формата. Режим наложения особенно эффективен при работе с длинными тестами, так как позволяет лучше видеть результаты распознавания.

Важно

Для наилучшего результата распознавания:

  • Убедитесь, что бланк хорошо освещен

  • Располагайте бланк параллельно камере
  • Используйте настройки камеры для улучшения качества изображения

Множественный выбор

О множественном выборе
Раздел множественного выбора специально разработан для проверки тестов, где на один вопрос может быть несколько правильных ответов. Это особенно полезно для сложных вопросов, требующих выбора нескольких вариантов или для вопросов с частично правильными ответами. В отличие от пакетной обработки, этот режим позволяет работать с тестами, где ученик может отметить более одного варианта ответа на каждый вопрос. Система также поддерживает проверку тестов, расположенных в два столбца, что особенно удобно для тестов с большим количеством вопросов.

4. Режимы проверки

Строгий режим

Вопрос считается правильным только если выбраны все правильные ответы и не выбраны неправильные

Частичный режим

Баллы начисляются пропорционально количеству правильно выбранных ответов

Совет

Строгий режим лучше использовать для вопросов, где все ответы должны быть выбраны правильно. Частичный режим подходит для вопросов с разным количеством правильных ответов.

6. Результаты проверки

Информация о результатах

Для каждого вопроса система показывает:

  • Выбранные учеником варианты ответов
  • Правильные варианты ответов
  • Количество правильно выбранных ответов
  • Процент правильных ответов
  • Балл за вопрос
Важные моменты
  • При использовании строгого режима ученик должен выбрать все правильные ответы для получения балла за вопрос
  • В частичном режиме баллы начисляются пропорционально количеству правильно выбранных ответов
  • Если ученик выбирает все варианты ответов, вопрос автоматически считается неправильным
  • Система учитывает как правильно выбранные, так и неправильно выбранные ответы
  • Для каждого вопроса можно установить разное количество правильных ответов
  • При использовании двухстолбцового формата система автоматически распределяет вопросы между столбцами
  • Для корректной работы с двухстолбцовым форматом важно обеспечить четкое разделение между столбцами на бланке
Рекомендации по использованию
  • Перед началом проверки убедитесь, что все правильные ответы отмечены корректно
  • Используйте строгий режим для вопросов, где важна точность всех ответов
  • Частичный режим лучше подходит для вопросов с разным количеством правильных ответов
  • Проверяйте результаты каждого теста перед отправкой в отчет
  • При необходимости можно изменить режим проверки и перезапустить обработку

Конструктор тестов

О конструкторе тестов
Конструктор тестов позволяет создавать собственные бланки OMR с настраиваемым количеством вопросов и вариантов ответов. Вы можете настроить внешний вид бланка и сохранить его в формате PDF или изображения. Система поддерживает создание как одностолбцовых, так и двухстолбцовых бланков, что особенно удобно для тестов с большим количеством вопросов.

1. Выбор формата бланка

Один столбец

Стандартный формат

Два столбца

Для длинных тестов

Совет

Двухстолбцовый формат идеально подходит для тестов с 20-50 вопросами. Это делает бланк более компактным и удобным для заполнения, при этом сохраняя хорошую читаемость.

2. Настройка текстовых элементов

  • Заголовок листа - название теста или работы
  • Описание - краткая инструкция или дополнительная информация
  • Поля для заполнения:
    • ФИО учащегося
    • Дата проведения
    • Предмет
    • Инструкция по заполнению

  • Скрытие кружков:
    • Кликните на кружок, чтобы скрыть его

Требования к бланку:

  • Оптимальное количество вопросов: от 5 до 20
  • Режим пакетной обработки работает только с тестами, где в каждом вопросе нужно выбрать один правильный ответ

  • При изменении количества вопросов нужно соответственно изменить количество строк
